EmberJS

Webpack Ruby Ember

Build a Simple app using EmberJS routes

1. Make sure Ember is on your computer.

To install Ember enter:
$ npm install -g ember-cli@2.14
That should have installed Ember.

2. Let's start a new project.

Get into the correct folder and type:
$ ember new ember-demo
Don't forget to get into that file.
$ cd ember-demo
Go ahead and open your project in an editor.

3. Let's change the file structure that Ember uses.

Go to the .ember-cli file. It should look like this now
{
/**
Ember CLI sends analytics information by default. The data is completely
anonymous, but there are times when you might want to disable this behavior.
Setting `disableAnalytics` to true will prevent any data from being sent.
*/
"disableAnalytics": false
}
We need to make it look like this
{
/**
Ember CLI sends analytics information by default. The data is completely
anonymous, but there are times when you might want to disable this behavior.
Setting `disableAnalytics` to true will prevent any data from being sent.
*/
"disableAnalytics": false,
"usePods": true
}

4. Let's use SCSS instead of CSS.

Go to the command line.
$ ember install ember-cli-sass
Now we need to rename the app/styles/app.css to app/styles/app.scss

Let's see if we have a good build

To the command line
$ npm start
If all went well you should see this at localhost:4200

Let's make some styles and routes.

1. Let's add some default styles

Create a new file in app/styles folder named _box-sizing.scss
Then add this:
* {
box-sizing: border-box;
}
html {
margin: 0;
}
body {
margin: 0;
}
ul {
padding: 0;
list-style: none;
}
a {
text-decoration: none;
&:hover {
text-decoration: underline;
}
}
.container {
max-width: 65rem;
margin: 0 auto;
}
img {
max-width: 100%;
max-height: 100%;
}
This file should be universal styles.
Now we need to link this to the app.scss file. Add this to the top.
@import 'box-sizing';

2. Now we need the application route.

The application route will be the our default view.
Anything here will be everypage. So top-navs and footers belong here.
$ ember g route application

3. Go to the new application template.

There you should see {outlet}}.
We will add our nav section before the {{outlet}}
In app/application/template.hbs add this(you will need to add all of the brackets)
nav class="nav"
div class="container"
div class="nav__content"
h1 class="nav__content-title"Ember Demo/h1
div class="nav__content-box"
a href="" class="nav__content-box-link"Page 1/a
a href="" class="nav__content-box-link"Page 2/a
a href="" class="nav__content-box-link"Page 3/a
/div
/div
/div
/nav
{{outlet}}

4. Now add the styles for the nav.

Create a new file in app/styles folder named _nav.scss
Go ahead and link this in the app.scss
@import 'nav';
In the nav.scss paste this:
.nav {
margin: 0;
background: #e77562;
&__content {
display: flex;
justify-content: space-between;
color: white;
&-box {
margin: 1.5rem;
display: flex;
&-link {
margin-left: 1rem;
color: white;
}
}
}
}

5. Check your build

If your server is not running go ahead and start it.
$ npm start
It should look like this
Hopefully that worked, if not check for typos then read some docs.

Now we should add the index route.

1. Create the route

Back to the command line
$ ember g route index

2. Write some HTML

Now go to app/index/template.hbs and enter this(you need to add the html tags)
div class="center"
h1 class="center__tag"This is the index/h1
/div

3. Create a new scss file

Create a new file in app/styles folder named _center.scss
Go ahead and link this in the app.scss
@import 'center';

4. Now add some style

Go to app/styles/center.scss and add
.center {
display: flex;
flex-direction: column;
justify-content: center;
height: 80vh;
&__tag {
text-align: center;
}
}

We need some more routes

1. We need to add some more routes.

Add a sidebar route with some nested routes.
$ ember g route sidebar
$ ember g route sidebar/option-one
$ ember g route sidebar/option-two
Let's add two more routes to give us more to link to.
$ ember g route about
$ ember g route contact

2. Let's change the a tags to link-to ember tags

Go to app/application/template.hbs it should look like this:
div class="container"
div class="nav__content"
h1 class="nav__content-title"Ember Demo/h1
div class="nav__content-box"
a href="" class="nav__content-box-link"Page 1/a
a href="" class="nav__content-box-link"Page 2/a
a href="" class="nav__content-box-link"Page 3/a
/div
/div
/div
/nav
{{outlet}}
Make it look like this instead. Don't forget the html brackets!
nav class="nav"
div class="container"
div class="nav__content"
{{#link-to "index" class="nav__content-box-link"}}h1Ember Demo/h1{{/link-to}}
div class="nav__content-box"
{{#link-to "sidebar" class="nav__content-box-link"}}Sidebar{{/link-to}}
{{#link-to "about" class="nav__content-box-link"}}About{{/link-to}}
{{#link-to "contact" class="nav__content-box-link"}}Contact{{/link-to}}
/div
/div
/div
/nav
{{outlet}}

3. Make sure the links work

You can check by adding something in each template that is unique.
Go to app/about/template.hbs and just add this to the top
Hello from the about page!
Go to app/contact/template.hbs and just add this to the top
Hello from the contact page!

Time for nested routes

1. Time for some markup

Go to app/sidebar/template.hbs and do this(add the brackets):
div class="bar"
div class="bar__sidebar"
h2 class="bar__sidebar-title"SideBar/h2
div class="bar__sidebar-tag"This will stay for all of the nested routes./div
{{#link-to "sidebar.index" class="bar__sidebar-link"}}This is the main or index link.{{/link-to}}
{{#link-to "sidebar.option-one" class="bar__sidebar-link"}}Here is the first option{{/link-to}}
{{#link-to "sidebar.option-two" class="bar__sidebar-link"}}Here is the second option{{/link-to}}
a href="https://emberjs.com/" class="bar__sidebar-link"EmberJS/a
a href="https://www.google.com/" class="bar__sidebar-link"Google/a
a href="http://www.bing.com/" class="bar__sidebar-link"Bing/a
{{#link-to "index" class="bar__sidebar-link"}}Back to Home{{/link-to}}
/div
div class="bar__content"
{{outlet}}
/div
/div
Go to app/sidebar/index/template.hbs(don't forget to add the brackets)
div class="center" h1 class="center__tag"This is the Sidebar Index Page/h1 /div
Now you can do the same for both of the other option pages. Just change the page names

2. Add our styles.

Create a new file at app/styles/sidebar.scss
Link that new file to the app.scss
@import 'sidebar';
Go back to the app/styles/sidebar.scss and add
.bar {
display: flex;
&__sidebar {
width: 25%;
padding: 0.5rem;
border-right: 1px solid black;
text-align: center;
&-tag {
margin-bottom: 1rem;
}
&-link {
display: block;
margin: 0.5rem;
}
}
&__content {
width: 75%;
}
}
