Webpack Ruby Ember

Build a Simple app using EmberJS

1. Make sure Ember is on your computer.

To install Ember enter:
$ npm install -g ember-cli@2.14
That should have installed Ember.

2. Let's start a new project.

Get into the correct folder and type:
$ ember new ember-demo
Don't forget to get into that file.
$ cd ember-demo
Go ahead and open your project in an editor.

3. Let's change the file structure that Ember uses.

Go to the .ember-cli file. It should look like this now
{
/**
Ember CLI sends analytics information by default. The data is completely
anonymous, but there are times when you might want to disable this behavior.
Setting `disableAnalytics` to true will prevent any data from being sent.
*/
"disableAnalytics": false
}
We need to make it look like this
{
/**
Ember CLI sends analytics information by default. The data is completely
anonymous, but there are times when you might want to disable this behavior.
Setting `disableAnalytics` to true will prevent any data from being sent.
*/
"disableAnalytics": false,
"usePods": true
}
