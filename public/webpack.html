Bundle a project using Webpack

This will show you how to bundle different modules together and have a large and complete js file to load on a script tag or wherever you deploy. For this exercise we will make a page that uses two different JS files along with some SCSS. We will also bundle jQuery and some other tools using npm.

1. Lets start a new project.

In your terminal type
$mkdir webpack-demo
Make sure you get into your new folder.
$cd webpack-demo

2. Now lets create a package.json file.

In your terminal type
npm init
This should prompt you to answer a few questions. You should be able to just hit enter for all. If you want to fill out the discription you can or you can do it later. The one important thing is the entry file name, by default it should be index.js. That is fine but you can change that too if you want. Finally it will show you the JSON and ask if it is okay, just hit enter.
